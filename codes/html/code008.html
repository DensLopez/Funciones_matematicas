Son aquellas que de manera exacta no pueden ser representadas por una suma finita de polinomios.
Ejemplos:

  • Logaritmos
  • Funciones trigonometricas
  • Exponenciales

Funciones trascendentes

Son funciones que no pueden ser expresadas con polinomios.

Funciones trigonomtricas

Son funciones que se relacionan con la geometra.

  • Seno = $\sin(x) = \frac{cateto \; opuesto}{hipotenusa}$

  • Coseno = $\cos(x) = \frac{cateto \; adyacente}{hipotenusa}$

#Funcin trigonometrica
def f(x):
  return np.cos(x)

#Declarando variables
N=100
x = np.linspace(-10,10, num=N)
y = f(x)

plt.plot(x,y,label=r'$\cos(x)$')
plt.grid()
plt.title(r'Funcin trigonometrica $f(x) = \cos(x)$',fontsize=23)
plt.xlabel(r'$x$',fontsize=18)
plt.ylabel(r'$f(x)$',fontsize=18)
plt.legend()
plt.show()

Funcin exponencial

Tienen la forma de $$f(x)=a^x$$ donde la base $a$ es una constante positiva. Un gran ejemplo de una funcin exponencial es usando la base como el nmero de euler:

$$f(x)=e^x$$

In[]:
#Funcin exponencial
def f(x):
  return np.exp(x)

y=f(x)

plt.plot(x,y,label=r'$f(x) = e^x$')
plt.grid()
plt.title(r'Funcin exponencial $f(x) = e^x$',fontsize=23)
plt.xlabel(r'$x$',fontsize=18)
plt.ylabel(r'$f(x)$',fontsize=18)
plt.legend()
plt.show()

Funcin logaritmo

El logaritmo est definido por la relacin:

$$log_{b}(x) = n \Longleftrightarrow x=b^n$$

donde:

  • $b$ es la base.
  • $n$ es el exponente al que est elevado la base.
  • $x$ es el resultado de elevar la base $b$ al exponente $n$

Ejemplo:

Teniendo b=2 y n=8, y sustituyendo en $b^n=x$ entonces:

$$2^8=256$$

Por lo que $x=256$. Calculando el logaritmo base 2 de $x$ es:

$$log_{2}(256) = 8$$

In[]:
#Funcin logaritmo base 2
def f(x):
  return np.log2(x)

x = np.linspace(0.001,256, num=1000)

plt.plot(x,f(x),label=r'$f(x) = log_2(x)$')
plt.grid()
plt.title(r'Funcin logaritmo $f(x) = log_2(x)$',fontsize=23)
plt.xlabel(r'$x$',fontsize=18)
plt.ylabel(r'$f(x)$',fontsize=18)
plt.legend()
plt.show()
