Funcin lineal

$$f(x)=mx+b$$

In[]:
# definiendo funcion parabola
def f(m,x,b):
  return m*x+b

#Definiendo variables
N = 1000
x = np.linspace(-5,5, num=N)

#Creando graficas
fig, ax = plt.subplots()
ax.plot(x,f(2,x,3),label=r'$f(x)=mx+b$',color='blue',linewidth=2)
ax.grid()
ax.axhline(y=0, color='r',linewidth=0.7)
ax.axvline(x=0, color='r',linewidth=0.7)
plt.title(r'Funcin lineal $f(x)=mx+b$',fontsize=23)
plt.xlabel(r'$x$',fontsize=18)
plt.ylabel(r'$f(x)$',fontsize=18)
plt.legend()
plt.show()

Para qu sirve?

Todo lo que pasa mediante esta funcin regresa con el mismo valor.Para qu sera bueno esto? Sirve para mantener valores durante un proceso, por ejemplo si tu quisieras mantener el valor de una venta que es a trves del proceso de regresin lineal, pues lo que te interesa al final, es obtener la prediccin de algn valor, es decir cuando sigues tendencias de algunos datos.

Funcin escaln o de Heaviside

$$H(x) = \begin{cases} 0, &\quad \text{para, } x < 0 \\ 1, &\quad\text{para. } x \ge 0 \\ \end{cases} $$

In[]:
def H(x):
  Y = np.zeros(len(x))
  for idx,x in enumerate(x):
    if x>=0:
      Y[idx]=1
  return Y
    

N=1000
x = np.linspace(-10,10, num=N)
y = H(x)
fig, ax = plt.subplots()
plt.plot(x,y,label=r'$H(x)$',color='blue',linewidth=2)
plt.grid()
ax.axhline(y=0, color='r',linewidth=0.7)
ax.axvline(x=0, color='r',linewidth=0.7)
plt.title(f'Funcin Heaviside\n$H(x)$\n',fontsize=23)
plt.xlabel(r'$x$',fontsize=18)
plt.ylabel(r'$f(x)$',fontsize=18)
plt.legend()
plt.show()

Todos los valores que sean menores que cero, pues los va a volver 0 y pues todos los valores mayores a 1, los va a volver 1.

Para qu sirve?

Para poder hacer clasificaciones categricas, esto nos podra indicar si algo est [prendido,apagado],[existe, no existe] es decir para jugar con valores binarios.
Esto en modelos de clasifican super sencillos es de gran utilidad.

Funcin ReLU

$$R(x)=max(0,x)$$

In[]:
def f(x):
  return np.maximum(x,0)
    

N=1000
x = np.linspace(-8,8, num=N)
y = f(x)

fig, ax = plt.subplots()
plt.plot(x,y,label=r'$R(x)=max(0,x)$',color='blue',linewidth=2)
plt.grid()
ax.axhline(y=0, color='r',linewidth=0.7)
ax.axvline(x=0, color='r',linewidth=0.7)
plt.title(r'$R(x)=max(0,x)$'+'\n',fontsize=23)
plt.xlabel(r'$x$',fontsize=18)
plt.ylabel(r'$R(x)$',fontsize=18)
plt.legend()
plt.show()

Para qu sirve?

Significa que si; $$R(x) = \begin{cases} 0, &\quad \text{para, } x < 0 \\ x, &\quad\text{para. } x \ge 0 \\ \end{cases} $$

En otras palabras para valores de $x<0;R(x)=0$ para valores de $x$ menores a $0$ la funcin vale $0$, por otro lado para valores de $x>0;R(x)$ para valores de $x$ mayores a $0$ la funcin vale $x, es decir va a empezar a existir un crecimiento.

A esta funcin se le pone al final de un perceptrn, cuando nosotros queremos simular un proceso de neuronas muertas (cuando los valores tienden a ser $0$ o negativos es decir no existentes para el proceso que nosotros realizamos con nuestro modelo y cuando tenemos un valor que si pueda procesar; es decir mayor a cero, entonces si vamos a tomar el valor a procesar).

Digamos que la neurona va decir: Existe? entonces lo tomo en cuenta, no existe? entonces lo pongo como cero.

Existe una variacin de esta funcin que se llama Leaky RELU; lo que hace es que en los valores donde est el 0, los pone con una ligera pendiente, para que exista una pequea ponderacin.
